# 📦 Sistema de Departamento e Funcionários

Projeto desenvolvido em Java com foco em Programação Orientada a Objetos (POO), simulando o gerenciamento de um departamento e seus funcionários.

---

## 🚀 Funcionalidades

* Cadastro de departamento
* Cadastro de funcionários
* Associação de endereço (email e telefone)
* Cálculo da folha de pagamento
* Exibição de relatório

---

## 🧠 Conceitos aplicados

* POO
* Encapsulamento
* Associação e composição
* Listas (`List`)

---

## 💻 Tecnologias

* Java
* Git & GitHub

---

git clone https://github.com/Leosalerno/EmpregadosOO.git
